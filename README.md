# Windows Server Active Directory Lab

## Project Overview
This project demonstrates the implementation of a Windows Server lab environment with Active Directory, DNS, and Group Policy configuration.

## Environment
- Windows Server 2008
- Active Directory
- DNS
- GPO
- VirtualBox Lab

## Objectives
- Deploy and configure Active Directory
- Configure DNS services
- Create and manage Group Policies
- Simulate corporate infrastructure environment

## Skills Demonstrated
- Windows Server Administration
- AD Management
- Infrastructure Setup
- Troubleshooting
- Network Configuration
